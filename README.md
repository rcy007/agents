# agents

Curated agent instructions (`AGENTS.md`, `CLAUDE.md`, `LINEAR.md`) for new projects.

## Usage

In any project directory:

```bash
npx @rcy007/agents@latest
```

Pick which files to drop in. Existing files prompt before overwriting.

The CLI fetches the latest version of each file from this repo's `main` branch at runtime — edits to the markdown files here are live for everyone immediately, no republish needed.
